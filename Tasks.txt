Web3-Scripts
The following are tasks we require you to complete in order to get a good grasp of what you can and cannot do using web3 and be able to contribute to your projects accordingly. All these tasks are to be done on the rinkeby network.

Write a web3 script to transfer ether from one address to another and also to display a balance of the 2 addresses in question before and after transfer.

Write a web3 script to interact with an ERC20 token contract and execute its transfer,transferFrom, approve, allowance and balanceOf functions.

Create a dynamic function that takes contract address, event name, start block, end block as input and then fetches the events of the contracts address defined. Also perform filtration on the events fetched.

Create a dynamic function that takes contract address, events name and subscribes to the event defined.

Create a function that takes tx-hash as input and outputs whether the transaction is confirmed or failed, if confirmed display the amount of tx_fees incurred.

Create a function that takes tx_hash of an erc20 transfer and determines how many erc20 tokens were transferred along with the sender and receiver.

Create a script to send a transaction with a very low gas price and then create a script to speed up or cancel that transaction.

Create a script that fetches all transactions executed by an address.

Create a script that fetches all erc20 token transfers of a particular token.

Create a script that calculates the current number of token holders of a particular token.
